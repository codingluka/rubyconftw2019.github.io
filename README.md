Ruby Conf Taiwan 2019
===

## Installation

Development environment requirements :

- Ruby 2.4.1
- middleman 4.2

```
$ gem install middleman
```

### Install dependencies
```
$ bundle install
```

### CSS framework
[Bulma: Free, open source, & modern CSS framework based on Flexbox](https://bulma.io/)

## Starting Server
```
$ middleman
```

## Deploy
```
$ middleman deploy
```
